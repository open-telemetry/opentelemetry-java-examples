# OTLP Example

## Prerequisites

* Java 17 or higher is required to run Gradle and build this example
* Java 8 or higher may be used to run the example once it has been built
* Docker Compose

## How to Run

Assuming you're on a Unix-like system (macOS, Linux, etc.):

In a shell:

```shell
cd docker
docker compose up
```

In a separate shell:

```shell
../gradlew run
```
